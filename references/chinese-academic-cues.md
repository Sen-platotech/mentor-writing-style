# 中文人文社科论文 · 文风识别线索

> 深读与合成 style-guide 时读本文。这里列的是中文社科论文里**反复出现、可观察、可计数**的风格载体；
> 它们是"线索清单"，不是必然规律——以目标作者语料中**实际出现**的为准，并回原文逐字坐实。

## 一、作者分档常识（triage 用）

中文社科一个文件夹常混入多种"作者关系"，必须先分清，否则学到的是混合体。

- **文件名/署名格式**常见模式：
  - `作者 - 年份 - 标题`：多为该作者独著/一作。
  - `甲和乙 - 年份 - 标题`：合著；**甲常为第一作者**。导师团队里，"导师和学生"或"学生和导师"都有，**谁主笔需问用户**。
  - `外国人名·译者名 - 年份 - 标题`、含"著""译""编译"字样：**译作**——正文是外国作者的论证，中文只反映**译者的术语选择**。
  - 首页作者块里出现"译者简介""本文原载《…》""译自"：同上，译作。
- **分档与权重**：独著/一作（最高）＞与晚辈合著（取结构、降语言）＞本人译作（仅取术语译名）＞他人/师承上源（单列或排除）。
- **时间加权**：作者高产、亲自操刀的年段权重更高；越晚近、越像"挂名带学生"的合著，语言权重越低。**主动问用户**人物关系与黄金年段。
- 怎么快速判断："谁是导师、谁是学生、哪几年是本人巅峰、哪些是译作"——**用户最清楚，先问再分**。

## 二、标题样态

- **正：副 结构**：`主标题：副标题`，副标题常点方法/视角，如"……：基于××的理论诠释/学理考察/分析框架"。
- **三元 / 四元并列正标题**（中文社科高频癖好）：如"观念、话语、修辞""发生路径、内在逻辑及意义评析""内在张力、多维特征与基本面向"。并列项**字数对齐、词性一致**。
- 看目标作者是否有**反复使用的标题句式**——这往往直通其"签名结构模板"。

## 三、摘要样态（常见两型）

- **评析/引介型**：[对象] 作为 [学科中的定位] → 从 [来源] 汲取 [基石/动力] → 在 [学理基础] 上展开 → 可从 [N个方面] 对其 [贡献、不足、展望] 综合审视。
- **反思/翻案型**：[对象] 推进了 [成就，常是三元短语] **，但** [指出盲点] → [已有学者认识到…] → [本文立场] 有助于从 [并列N方面] 把握 [趋势]。
  - "**但/然而**"是这型摘要的**结构枢纽**：从"肯定"翻到"问题"。留意作者用哪个转折词、转折落点在哪。

## 四、谋篇布局

- 期刊论文骨架：`一、二、三、（四、）+ 结语`；其下 `（一）（二）` 或 `1. 2. 3.`。
- 学位论文：绪论四件套（选题意义 / 文献综述 / 方法与结构 / 创新与不足）+ 正文分章。
- **偏好"三分/四分"**：把对象拆成三或四个并列维度（如本质=A/B/C，特征=甲/乙/丙）。数一数作者每层爱拆几份。
- **签名结构模板**：若**多篇文章章节安排同构**（同样的几个一级标题反复出现），这就是最值钱的发现——**拆到小节粒度**（连每节内部的"1./首先"层次都对齐），并给"填空式套用示意"。

## 五、段落与论证肌理

- **段落三拍**：主题句（亮判断）→ 排比分述 → 小结/推断。
- **分述排比**：`首先 / 其次 / 再次 / 复次 / 最后`；`其一 / 其二 / 其三`；`就……而言`。看作者惯用哪一套、每段铺几项。
- **下定义**：关键概念常以"**……是指，……**"硬定义后再展开。记录作者怎么给术语下定义。
- **对立词对法**（切分议题利器，"变迁/类型/路径"类主题尤甚）：先列 N 个划分角度，再对应 N 组对立词对逐对考察，如"根据…这四个角度，从 外生/内生、强制/诱致、激进/渐进、合意/意外 四个方面分别考察"。
- **判断的分寸**：留意是强断言还是多用 `往往 / 一定程度上 / 大体 / 总体而言`；是否给**双向评价**（既肯定又保留）。
- **过渡词库**（可计数，做频次表）：`具体而言、在此意义上、这意味着、由此可（见/推断）、从……来看、值得注意的是、一方面…另一方面、不仅…还、尽管…但是、相比…更为`。

## 六、用词与术语

- **骨架词/口头禅**：用 `corpus_stats.sh` 跨篇计数，挑高频者作为"必用词"（如某作者反复出现的"取向/张力/路径依赖/谱系/机制/建构"等）。**用数字定，别凭印象。**
- **三元短语癖**：作者自造的并列短语（字数对齐、词性一致），列一份"仿写清单"。
- **术语译名**：从其译作与正文里固定下来的中译（及对应英文原词），供写作时统一；引介新词时按其习惯"先中译再括注英文"。

## 七、引证体例

- 注释形式：脚注还是尾注；英文文献是否给全称（作者、文题、刊名卷期年页）；中译本是否注译者与出版社；正文夹注样式（如 `〔n〕页码`、`[n](P页)`）。
- **对话对象**：作者偏重引哪类来源（英文原典？本国经典？同门/本校学脉？）——引介型文章是否**主动接上国内既有讨论**，而非只谈西方。这本身是其学术立场的一部分。

## 八、提醒

- 以上是"看什么"，不是"一定有什么"。**目标作者没有的，就不要写进指南。**
- 每条特征在 style-guide 里都要配**一句原文引文 + 篇名年份**，并 `grep` 逐字核对（注意 pdftotext 会在 CJK 间插空格，匹配前先 Unicode 去空白）。
