# 文风 style-guide —— 罗承宇（虚构）· 历史社会学 · 国家建构与基层治理

> 一句话用法：写作时加载本文件；动笔前对照 §3 模板与 §7 Do/Don't，收尾用 §8 清单自检。
> 本指南（示例）基于"深读 11 篇核心语料"归纳，特征均配原文引文【篇名·年份】——**示例中的引文皆为虚构**。

## 0. 用法与语料档位
- 目标：能"用罗承宇的口吻"逐段写作，而非泛泛模仿。
- 权重：独著（最高，全仿）＞与学生合著（取结构、降语言）＞本人译作（仅取译名）＞他人/师承上源（单列）。叠加时间加权：2008–2017 独著为黄金段。

## 1. 一段话文风画像（描述性）
罗承宇的论文是典型的**"长时段—机制—回响"体**：不就事论事，而是把一项制度或治理形态放回**历史结构与社会经济条件**中，先讲它"怎么长出来"，再拆"靠什么运转"，最后问"今天还剩什么"。语言沉稳、偏长句，爱用**时序与因果连词**（"循此逻辑""究其根本"），爱把论点拆成**自上而下／自下而上**这类对偶，判断克制，常以"质言之"收束一段。例证以**史料与比较个案**为主，少抽象思辨。

## 2. 谋篇布局
### 2.1 标题公式
- 正：副 结构，副标题点个案或时段：`……：以华北某县（1900–1949）为例`。
- 三元并列正标题，如《X 的历史生成、运作机理与现实回响》。
### 2.2 文章骨架
`引言 → 一、二、三 → 余论`；每节下 `（一）（二）（三）`。偏好**三分**。
### 2.3 摘要公式
- **机理型**：[对象] 在 [历史条件] 下生成 → 经 [机制] 得以运作与再生产 → 对 [当代] 仍有 [回响]。
- **翻案型**：既有研究多以 [流行解释] 视 [对象]，**然而** [史料显示别样图景] → 本文主张从 [视角] 重审。
  > "然而"是翻案型摘要的枢纽。〔示例·虚构〕

## 3. ⭐签名结构模板拆解：《X 的历史生成、运作机理与现实回响》
> 罗承宇介绍任一制度/治理形态都套此骨架（保甲、漕运、单位制……）。可直接改写到你的题目上。

**一、历史生成**（讲"怎么长出来"）— 三小节
1. 结构前提（社会经济与人口条件）
2. 触发时刻（关键事件/危机）
3. 路径锁定（为何此后难以逆转）

**二、运作机理**（讲"靠什么转"）— 三小节
1. 行动者与激励（谁在其中、图什么）
2. 制度装置（规则、文书、考核等具体抓手）
3. 再生产机制（如何自我维系）

**三、现实回响**（讲"今天还剩什么"）— 三小节
1. 当代延续（制度遗产）
2. 张力与局限
3. 本土启示

### 填空式套用示意（把"X"换成你的对象）
```
一、X 的历史生成
   1. 结构前提  2. 触发时刻  3. 路径锁定
```

## 4. 段落与论证肌理
### 4.1 段落三拍
主题句（亮判断）→ 史料/个案分述 → "质言之/究其根本"小结。
### 4.2 连接词库（按"实测频次"，示例）
`究其根本 / 循此逻辑 / 就此而言 / 不难发现 / 质言之`；`一方面…另一方面`；`不仅…更`；`与其说…不如说`。
### 4.3 下定义法
"**所谓 X，乃是指……**"硬定义后再展开。〔示例·虚构〕
### 4.4 对偶切分法
用对偶轴切议题：`自上而下／自下而上`、`中心／边缘`、`连续／断裂`、`正式／非正式`——"从 A 与 B 两个方向分别考察"。
### 4.5 判断的分寸
多用 `大体 / 在相当程度上 / 未必 / 恐怕`；常给双向评价（"既……亦……"）。

## 6. 引证与对话对象
- 脚注；英文文献全称；史料注明出处与卷页；正文夹注 `（页码）`。
- 偏好历史社会学经典（韦伯式比较、国家建构与基层治理研究）与本国史料，并**主动接上国内史学/社会学讨论**。

## 7. Do / Don't
**Do**：放回长时段与社会经济条件立论；先生成、后机理、再回响；史料/个案说话；论点拆成对偶轴；段末"质言之"收口。
**Don't**：不抽象空谈；不下绝对断言；不堆叠时髦术语；不只谈西方而忽略本土史料与讨论。

## 8. 开工清单（写每节前自检）
1. 标题能否做成"正：副（个案/时段）"或三元并列？
2. 新概念是否已"所谓……乃是指……"下定义？
3. 是否落在"历史生成→运作机理→现实回响"的对应位置？
4. 论点是否拆成了对偶轴并双向考察？
5. 是否有史料/个案支撑，而非空论？
6. 判断是否留有余地？段末是否有"质言之"式小结？
7. 是否接上了国内既有讨论？
