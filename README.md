# mentor-writing-style · 导师文风提炼

> 上传一位学者（通常是你导师）的若干论文，自动学其文风，产出一份**既读得懂、又能照着写**的写作风格指南（style-guide）。**专为中文人文社科论文设计。**
>
> *A Claude skill for the humanities & social sciences: learn a scholar's academic writing style from their corpus and distill it into an operational style guide.*

## 为什么是人文社科，为什么可行

一位成熟学者的研究，往往是**可重复、可追踪、可预测、可复现**的：

- **可重复**——他在不同主题上反复使用同一套章节骨架与论证套路；
- **可追踪**——这些习惯在历年论文里留下连续、可考的轨迹；
- **可预测**——读过若干篇后，你大致能料到下一篇怎么起、怎么转、怎么收；
- **可复现**——正因为上述规律稳定，它们就能被**提炼成模板，再被照着重新写出来**。

人文社科尤其如此：这里的学术个性沉淀在**语言、结构与概念用法**中，而非实验数据里。本 skill 正是抓住这种"风格的规律性"，把"导师的味道"从可意会变成**可操作**——给出标题/摘要公式、签名章节模板、连接词与高频骨架词、下定义与论证套路、引证体例，最终汇成能直接驱动写作的指南。

## 它特别注意的三件事

1. **分清谁的声音。** 中文社科论文常把导师独著、师生合著、本人译作、乃至师承上源的文章混在一个文件夹里。本 skill 先做**作者分档**并按"本人声音"加权，避免学成一锅大杂烩。
2. **用数字而非印象。** 高频词、连接词都**实际计数**，不靠"我觉得他常说……"。
3. **能用，不只是赏析。** 产出含填空式模板、词库、Do/Don't 与开工清单，每条特征都回原文**逐字核对**、附出处。

## 怎么用

1. 把目标作者的论文（PDF/DOCX/TXT）放进一个文件夹。
2. 对 Claude 说，例如：「学习这个文件夹里我导师的写作风格，总结成可套用的规范」。
3. Claude 会：列清单 → **作者分档并请你确认**（谁是导师/学生、哪几年是本人巅峰）→ 抽取文本 → 脚本量化扫描 → 精读高权重篇目 → 产出 `<作者>文风_style-guide.md` → 回原文逐字核对。

> 提示：文件名带上 `作者 - 年份 - 标题` 更利于自动分档；师生关系、黄金年段直接告诉 Claude 最准。

## 你会得到什么

一份 `style-guide.md`（结构见 `references/style-guide-template.md`），含：文风画像、标题/摘要/谋篇公式、**签名章节模板的逐小节拆解 + 填空式套用**、连接词库、**实测词频表**、术语译名、引证体例、Do/Don't、开工清单，以及诚实标注"学的是谁的声音"的**作者分档表**。

`examples/` 里有一份**完全虚构的演示示例**（虚构学者"罗承宇"·历史社会学），用于展示输出的形态与密度——其中人名、篇名、引文皆为编造，仅作样例。

## 局限

- 面向**中文人文社科**论文调校；其他语言/学科可用但提示未必贴合。
- 风格规律性强不等于"千篇一律"：越是创新性、跨界的作者，可预测性越低，提炼难度越大。
- 扫描版（图片型）PDF 无文本层，需 OCR；若本人声音的文本篇目已足够，默认跳过并注明。
- **作者关系与黄金年段以你的判断为准**——请在分档环节校正。
